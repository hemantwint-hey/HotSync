# 🚀 HotSync

HotSync is a real-time chat and collaboration application developed using ASP.NET Core. It enables users to communicate instantly through secure authentication and real-time messaging powered by SignalR. The project is designed with a clean architecture, RESTful APIs, and SQL Server for efficient data management.

### Tech Stack
- ASP.NET Core
- C#
- SignalR
- Entity Framework Core
- SQL Server
- JWT Authentication
